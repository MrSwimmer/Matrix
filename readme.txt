Класс матриц в C++: matrix

----------------------------------------------------------

Подключение заголовчного и исполнительного файлов класса

1.Прикрепить к проекту файл matrix.h

2.Прикрепить к проекту файл matrix.cpp

3.В файле main.cpp подключить заголовочный файл matrix.h
#include "matrix.h"

----------------------------------------------------------

Описание класса

1.Перегружен оператор присваивания =
Если A и В объекты класса матриц, А=B (матрица А принимает значения матрицы В)

2.Перегружен оператор сложения +
Если A и В объекты класса матриц, А+B (А+В матрица, полученная путем сложения матриц)

3.Перегружен оператор вычитания -
Если A и В объекты класса матриц, А-B (А-В матрица, полученная путем вычитания матрицы В из А)

4.Перегружен оператор умножения *
Если A и В объекты класса матриц, А*B (А*В матрица, полученная путем умножения матриц)

5.Метод нахождения определителя матрицы float det(matrix c)
Нахождение определителя  матрицы с, c.det(c)

6.Метод нахождения союзной(присоединенной) матрицы matrix adjm(matrix c)
Нахождение союзной матрицы с, c.adjm(c)

7.Метод нахождения транспонированной матрицы matrix transm(matrix c)
Нахождение транспонированной матрицы с, c.transm(c)

8.Метод умножения матрицы на скаляр matrix skal(matrix c, float n)
Умножение матрицы с на число n, c.skal(c)

9.Метод нахождения обратной матрицы rvrsm(matrix c)
Нахождение обратной матрицы с, c.rvrsm(c)

10.Метод возведения матрицы в степень powm(matrix c, int n)
Возведение матрицы с в степень n, c.powm(c,  n)

11.Конструктор матрицы matrix()
Создание матрицы с, matrix c()

12.Конструктор матрицы с заданным размером matrix(int n, int m)
Создание матрицы с размером nxm, matrix c(n,m)

13.Метод вывода матрицы на экран void printMatrix()
Вывод матрицы с на экран, с.printMatrix()

14.Деструктор матрицы c, ~matrix()
